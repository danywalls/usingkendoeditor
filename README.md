# Effortlessly Create Rich Text Content with Angular Kendo UI Editor

## Project Summary
This project showcases how to implement a rich text editor in Angular using Kendo UI, including toolbar customization and security best practices.

## How to Run
### Installation
1. `npm install` 

### Running
1. `ng serve` 
2. Interact with the rich text editor to format content and bind it to HTML.
